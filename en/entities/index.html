    Entities

    Client

    Description

    The class responsible for the client required for authorization when receiving the token, as well as using the Service.

    Properties

    Property Type Description
    user_agent str The user ID in the system.
    client_id str Unique identifier of the client.
    client_secret str Client's secret key.

    Methods

    __init__(user_agent, client_id, client_secret)

    Designer. Creates a new instance.

    Input parameters:

    Parameter Type Description
    user_agent str User ID.
    client_id str Unique identifier of the client.
    client_secret str Client's secret key.

    TokenReceiver

    Description

    The class responsible for token acceptance and authorization.

    Properties

    Property Type Description
    _\ _login str Username of the user.
    __password str User password.
    client str Client application.
    __token str User access token.

    Methods

    __init__(login, password, client="Kate")

    Designer. Creates a new instance.

    Input parameters:

    Parameter Type Description
    login str User login.
    password str User password.
    client str Client name (by default "Kate").

    auth()

    Performs authorization using the provided username and password. If necessary, it interacts with an SMS code or a captcha.

    Return value

    Type Description
    bool Boolean value indicating whether authorization is successful.

    get_token()

    Outputs the token in the console (if authorization was successful).

    save_to_config()

    Saves the token and user data in the configuration file (if authorization was successful).

    create_path(filename)

    Creates the path to the file.

    Input parameters:

    Parameter Type Description
    filename str File name.

    Return value

    Type Description
    str Path to the file.

    Service

    Description

    A class designed to work with music services.

    Properties

    Property Type Description
    user_agent str The user ID in the system.
    __token str User access token.

    Methods

    __init__(user_agent, token)

    Designer. Creates a new instance of the service.

    Input parameters:

    Parameter Type Description
    user_agent str User ID.
    token str Access token.

    parse_config(filename = "config_vk.ini")

    Creates an instance of the service based on data from the configuration file.

    Input parameters:

    Parameter Type Description Default value
    filename str Configuration file name "config_vk.ini"

    Return value:

    Type Description
    Service An instance of the service.

    get_count_by_user_id(user_id)

    Gets the number of the user's songs.

    Input parameters:

    Parameter Type Description
    user_id Union[str, int] VK user ID.

    Return value:

    Type Description
    int The number of songs of the user.

    get_songs_by_userid(user_id, count = 100, offset = 0)

    Search for songs by VK user ID.

    Input parameters:

    Parameter Type Description Default value
    user_id Union[str, int] VK user ID.
    count int Number of results (maximum 100 for VK API). 100
    offset int Offset of results. 0

    Return value:

    Type Description
    List[Song] List of Song objects.

    get_songs_by_playlist_id(user_id, playlist_id, access_key, count = 100, offset = 0)

    Getting songs by playlist ID VK.

    Input parameters:

    Parameter Type Description Default value
    user_id Union[str, int] VK user ID.
    playlist_id int Playlist ID VK.
    access_key str VK playlist access key.
    count int Number of results (maximum 100 for VK API). 100
    offset int Offset of results. 0

    Return value:

    Type Description
    List[Song] List of Song objects.

    get_songs_by_playlist(playlist, count = 10, offset = 0)

    Getting songs by playlist instance.

    Input parameters:

    Parameter Type Description Default value
    playlist Playlist A playlist instance.
    count int Number of results. 10
    offset int Offset of results. 0

    Return value:

    Type Description
    List[Song] List of Song objects.

    search_songs_by_text(text, count, offset)

    Search for songs by query text.

    Input parameters:

    Parameter Type Description Default value
    text str Request text.
    count int Number of results. 3
    offset int Offset of results. 0

    Return value:

    Type Description
    List[Song] List of Song objects.

    get_playlists_by_userid(user_id, count = 5, offset = 0)

    Getting playlists by VK user ID.

    Input parameters:

    Parameter Type Description Default value
    user_id Union[str, int] VK user ID.
    count int Number of results (maximum 100 for VK API). 5
    offset int Offset of results. 0

    Return value:

    Type Description
    List[Playlist] List of Playlist objects.

    search_playlists_by_text(text, count, offset)

    Search for playlists by query text.

    Input parameters:

    Parameter Type Description Default value
    text str Request text.
    count int Number of results. 5
    offset int Offset of results. 0

    Return value:

    Type Description
    List[Playlist] List of Playlist objects.

    save_music(song)

    Saving a song to a local disk.

    Input parameters:

    Parameter Type Description
    song Song Instance of the Song object.

    Return value:

    Type Description
    str Относительный путь к файлу.

    Playlist

    Описание

    Класс, представляющий музыкальный плейлист.

    Свойства

    Свойство Тип Описание
    title str Название плейлиста.
    count int Количество треков.
    owner_id int Идентификатор владельца.
    playlist_id int Идентификатор плейлиста.
    access_key str Ключ доступа к плейлисту.

    Методы

    __init__(title, count, owner_id, playlist_id, access_key)

    Конструктор. Создает новый экземпляр плейлиста.

    Входные параметры:

    Параметр Тип Описание
    title str Название плейлиста.
    count int Количество треков.
    owner_id int Идентификатор владельца.
    playlist_id int Идентификатор плейлиста.
    access_key str Ключ доступа к плейлисту.

    to_dict()

    Преобразует плейлист в словарь.

    Возвращаемое значение:

    Тип Описание
    dict Словарь с данными.

    from_json(item)

    Создает экземпляр плейлиста из словаря JSON.

    Входные параметры:

    Параметр Тип Описание
    item dict Словарь с данными JSON.

    Возвращаемое значение:

    Тип Описание
    Playlist Экземпляр плейлиста.

    Song

    Описание

    Класс, представляющий музыкальную композицию.

    Свойства

    Свойство Тип Описание
    title str Название композиции.
    artist str Исполнитель композиции.
    duration int Длительность композиции (секунды).
    track_id str Идентификатор трека.
    owner_id str Идентификатор владельца.
    url str URL для скачивания композиции.

    Методы

    __init__(title, artist, duration, track_id, owner_id, url = "")

    Конструктор. Создает новый экземпляр композиции.

    Входные параметры:

    Параметр Тип Описание
    title str Название композиции.
    artist str Исполнитель композиции.
    duration int Длительность композиции (секунды).
    track_id str Идентификатор трека.
    owner_id str Идентификатор владельца.
    url str URL для скачивания композиции.

    __str__()

    Возвращает строковое представление композиции в формате "Название - Исполнитель".

    Возвращаемое значение:

    Тип Описание
    str Строковое представление композиции.

    to_dict()

    Преобразует композицию в словарь.

    Возвращаемое значение:

    Тип Описание
    dict Словарь с данными.

    safe(song)

    Создает безопасную (без специальных символов) копию композиции.

    Входные параметры:

    Параметр Тип Описание
    song Song Экземпляр композиции для копии.

    Возвращаемое значение:

    Тип Описание
    Song Безопасная копия композиции.

    from_json(item)

    Создает экземпляр композиции из словаря JSON.

    Входные параметры:

    Параметр Тип Описание
    item dict Словарь с данными JSON.

    Возвращаемое значение:

    Тип Описание
    Song Экземпляр композиции.
