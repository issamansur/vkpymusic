    vkpymusic.Service

    A class for working with VK API.

    Attributes:

    Name Type Description
    user_agent str

    User agent string.
    __token str

    Token for VK API.
    logger Logger

    The logger for class.

    Example usage:

    >>> service = Service.parse_config()
>>> songs = service.search_songs_by_text("Imagine Dragons")
>>> for song in songs:
...     Service.save_music(song)
